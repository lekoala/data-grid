Data Grid

Standalone data grid web component

  • Default
  • Responsive
  • Server
  • Actions
  • API

Url should point to a json file that return a plain array or a json response with a data key that contains an array

  • sticky to enable stick headers and footers (usable with fixed height)
  • expand=true to allow cell content to spawn over multiple lines
  • resizable=true to make resizable columns
  • autosize=true to compute column sizes automatically
  • sort=true to make sortable columns
  • selectable=true to allow multi-row selections
  • single-select=true to allow single-row selection - no need to set selectable=true
  • save-state=true to save grid state

Fixed height with basic data. This looks odd if you set a height bigger than the number of rows per page. It works better with sticky.

  • transform Can transform row content
  • format Use custom formatting (eg: display a flag, use custom element for number formatting)
